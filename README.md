# Setup

## Python

    brew install python3

## Clone the notebook

    mkdir -p ~/projects
    cd ~/projects
    git clone git@github.com:clirdlf/juypter_notebooks.git

## Install Dependencies
    cd ~/projects/juypter_notebooks
    pip3 install -r requirements.txt

## Run the notebook

    jupyter notebook
